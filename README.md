# NODE.JS API 
## A Simple Node.js ``` API ``` Usingn Express.js And MongoDB

## version
```json
  "version": "1.0.0",
```

## dependencies 

| dependencies              | Version     |
| ----------------          | ----------- |
| Express.js                | 4.17.1      |
| Mongoose                  | 6.0.13      |
| path                      | 0.12.7      |


## devDependencies

| devDependencies           | Version     |
| ----------------          | ----------- |
| Nodemon                   | 2.0.15      |


## Scripts

```json
"scripts": {
    "start": "nodemon app.js"
  }
```
## Installation

```shell
$ git clone https://github.com/simo-oumaandi/node-api.git # or clone your own fork
$ cd node api
$ npm install
```
## Run App
```shell
$ npm run start
```
## Listing
```shell
$ http://localhost:3000/
```
